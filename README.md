# Hunch It — 互動原型 Demo

一支純靜態、單一 HTML、無後端、無外部請求的互動原型。線上版由 GitHub Pages 提供:
**https://omnis-labs.github.io/hunch-it-demo/**

「你現在卡在哪一種決定?」一支 app 收斂三條決策 flow:

1. **外來想法能不能變成我的交易** — 帶入 KOL／社群內容,拆開證據與假設,形成自己的買／等／放棄判斷。
2. **好公司現在是不是好交易** — 檢查現價已反映多少期待,以及上行是否值得下行風險。
3. **我的 Portfolio 實際上在押什麼** — 穿透 ETF 與共同假設,比較維持、減碼與換倉。

每條走 A→B→C 完整流程,含真實來源連結與監控。

> 定位:候選方案原型,**非產品決策**。核心哲學是「濾網不是標籤」——Hunch **永不下買賣指令**,
> 只描述狀態、讓你的判斷可見。示意數字非即時行情,不構成投資建議。

## 邊界

1. 不給進場價、停損、倉位或個人財務建議。
2. 只描述狀態,不下「買 / 不買」指令。
3. 所有立場句由使用者擁有——產品讓你的判斷可見,不替你判斷。
